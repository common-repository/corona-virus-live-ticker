== Description ==

This plugin displays the actual information about Covid-19.

You can see how many patient(s) are confirmed, recovered or dead, in the world, and the country you select.

[Demo](https://wp.webwapstudio.com/)

== API Information ==

API website: https://covid19.webwapstudio.com/

Privacy Policy and Terms and Conditions of Service: https://covid19.webwapstudio.com/privacy.html

= Features =
* On/Off target pages (top or bottom)
  * Home Page
  * Front page
  * All Individual Pages
  * All Individual posts
  * All Category pages
  * All Search pages
  * Or add shortcode to one or more pages
* On/Off display label
  * Change Label Text
  * Change Label Color
* On/Off displayed information
  * Display New Confirmed
  * Display New Deaths
  * Display New Recovered
* Select border Color
* Select desktop speed
* On/Off for mobile devices
* Select Mobile speed
* Order displayed info by
  * Country name (Ascent/Descent)
  * New Confirmed (Ascent/Descent)
  * New Recovered (Ascent/Descent)
  * New Deaths (Ascent/Descent)
* Select  one or more countries
* Edit More Info Links for each country.

== Installation ==

### Install from within WordPress

1. Visit the plugins page within your dashboard and select 'Add New'
2. Search for 'Corona Virus Live Ticker'
3. Click button 'Install'
4. Activate 'Corona Virus Live Ticker' from your Plugins page
5. Go to Corona Virus Live Ticker "Settings" tab

### Install manually

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Corona Virus Live Ticker "Settings" tab
